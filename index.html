HTML Tags explained

  1. <!DOCTYPE html>
    2. Specifies the standard: The DOCTYPE informs the browser about the specific HTML version (like HTML5) the page is written in.
  2. <html>
    3. This is the HTML tag. It defines the structure of the document such as headings, paragraphs, lists, and tables.
  3. <head>
    4. This is the head tag. It instructs the web browser on how to interpret and render the page, including character sets, styling, and script execution
  4. <title>Sabaidee </title>
    5. This is the title tag. It appears in the tab at the top of the browser window for each open page and helps with search engines like Google.
  5. </head>
    6. This closes the <head> tag
  6. <body>
    7. The body tag in HTML defines the main content of an HTML document using a combination of header and paragraph tags, and in this case, greeting you; the world!
  7. <h1>Hello (Sabaidee) World!</h1>
  8. </body>
    9. This closes the <body> tag
  9. </html>
    10. This closes the </html>

    Recap: And in the wink of an eye, we've learned a little bit of HTML and the Lao language.
